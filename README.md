# How to use

prebuild .exe in build.zip

or build from source

# build

## Libraries used in this project:
- glad
- glfw
- glm
- khr
- stb

## linux

### System requirements:
- cmake
- make
- libglfw3-dev

```
mkdir build
cd build
cmake ..
make
```

## windows

### System requirements:
- CMake
- VS

```
mkdir build
cd build
cmake ..
cmake --build . --config Release
```
